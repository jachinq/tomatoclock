# TomatoClock

## 项目：番茄钟

为了配合番茄钟工作法，写了一个倒计时应用。具备好看的UI设计。

支持功能：

- [x] 倒计时
- [x] 自动休息
- [x] 背景音乐
- [x] 倒计时结束通知：通过 Web Notification 提供提醒功能
- [x] 数据统计：按日、周、月、年几个维度进行数据统计，支持明细数据查看和简单修改
- [x] 明亮/黑暗主题切换
- [x] 设置功能
  - [x] 时钟设置：专注时间、休息时间设置
  - [x] 样式设置：设置随机图片背景及自定义随机背景API、设置开启一言
  - [x] 通知设置：设置开启/关闭结束通知、设置提醒音效、设置提醒图片
  - [x] 音乐设置：支持设置背景音乐开启/关闭，支持自定义音乐背景
  - [x] 数据操作：支持导出数据备份、支持导入备份数据



## 使用技术

- React
- mitt
- Vite
- tailwindcss



## 安装和启动

安装依赖

```
pnpm install
```

启动项目

```
pnpm run dev
```

构建项目

```
pnpm run build
```

访问：http://localhost:5173/
